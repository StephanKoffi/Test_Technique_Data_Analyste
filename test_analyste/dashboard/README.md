# Dashboard

## Description
Ce dossier contient les scripts nécessaires pour la création des tableaux de bord basés sur les données nettoyées.

## Structure
- **scripteDashboardGlobale.py** : Script principal pour générer les tableaux de bord globaux.

## Instructions
1. Assurez-vous que les données nettoyées sont disponibles dans le dossier `data/nettoyees/`.
2. Exécutez le script principal :
   ```bash
   python scripteDashboardGlobale.py
   ```
