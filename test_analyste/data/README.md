# Data

## Description
Ce dossier contient les données utilisées dans le projet, organisées en deux sous-dossiers :

- **brutes/** : Données brutes initiales.
- **nettoyees/** : Données nettoyées prêtes à l'analyse.

## Structure
- **brutes/** :
  - `centres_service.csv`
  - `demandes_service_public.csv`
  - `details_communes.csv`
  - `developpement.csv`
  - `documents_administratifs_ext.csv`
  - `donnees_socioeconomiques.csv`
  - `logs_activite.csv`
  - `reseau_routier_togo_ext.csv`

- **nettoyees/** :
  - `cleaned_centres_service.csv`
  - `cleaned_demandes_service_public.csv`
  - `cleaned_details_communes.csv`
  - `cleaned_developpement.csv`
  - `cleaned_documents_administratifs.csv`
  - `cleaned_donnees_socioeconomiques.csv`
  - `cleaned_logs_activite.csv`
  - `cleaned_reseau_routier.csv`

## Instructions
- Les données brutes doivent être placées dans le dossier `brutes/`.
- Les données nettoyées générées par les scripts de nettoyage seront enregistrées dans le dossier `nettoyees/`.
