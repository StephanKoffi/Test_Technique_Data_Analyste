# EDA (Exploratory Data Analysis)

## Description
Ce dossier contient les scripts pour effectuer une analyse exploratoire des données (EDA).

## Structure
- **eda_centres.py** : Analyse des données des centres de service.
- **eda_reseau_routier_togo_ext.py** : Analyse des données du réseau routier.
- **edaDemandeService.py** : Analyse des demandes de service public.
- **edaDetailCommunes.py** : Analyse des détails des communes.
- **edaDeveloppement.py** : Analyse des données de développement.
- **edaDocumentAdministrative.py** : Analyse des documents administratifs.
- **edaDonneesSocioEconomiques.py** : Analyse des données socio-économiques.
- **edaLog.py** : Analyse des logs d'activité.

## Instructions
1. Assurez-vous que les données nettoyées sont disponibles dans le dossier `data/nettoyees/`.
2. Exécutez les scripts correspondants pour analyser les données spécifiques.
