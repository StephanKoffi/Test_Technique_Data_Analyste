# Nettoyage

## Description
Ce dossier contient les scripts pour le nettoyage des données brutes afin de les rendre exploitables pour l'analyse et la création de tableaux de bord.

## Structure
- **scrypteNetoyageGlobal.py** : Script principal pour effectuer le nettoyage global des données.

## Instructions
1. Placez les données brutes dans le dossier `data/brutes/`.
2. Exécutez le script de nettoyage global :
   ```bash
   python scrypteNetoyageGlobal.py
   ```
3. Les données nettoyées seront enregistrées dans le dossier `data/nettoyees/`.
